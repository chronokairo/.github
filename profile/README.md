# ChronoKairo

Software house especializada em sistemas complexos, arquitetura de software e automação de processos de negócio.

---

## O que fazemos

Enquanto ferramentas de geração de código resolvem o trivial, construímos o que exige raciocínio de engenharia: sistemas distribuídos, integração com ecossistemas fiscais brasileiros, automação de operações críticas e plataformas B2B com requisitos não convencionais.

Nosso diferencial não é velocidade de entrega — é a capacidade de arquitetar soluções que funcionam quando os requisitos são difíceis, o ambiente é regulado ou a margem para erro é baixa.

---

## Áreas de atuação

**Sistemas e plataformas**
Aplicações web e desktop com lógica de negócio complexa, multi-tenant SaaS, ERPs customizados e sistemas offline-first para operações sem dependência de conectividade contínua.

**Automação e integrações**
Automação de processos via WhatsApp Business, integração com marketplaces, ERPs de terceiros, bancos e gateways de pagamento. Conectamos sistemas que não foram feitos para se conectar.

**Fiscal e compliance brasileiro**
Emissão de NF-e, NFC-e e NFS-e, comunicação com SEFAZ, SPED Fiscal e Contribuições, certificados A1/A3. Construímos sobre as especificações técnicas, não sobre wrappers frágeis.

**Arquitetura distribuída**
Sistemas P2P, sincronização offline, CRDTs, criptografia aplicada e modelos de consenso para ambientes sem servidor central.

## Como trabalhamos

Projetos sob medida, sem metodologia imposta. Entregamos o que foi acordado dentro do prazo, documentado o suficiente para não criar dependência, e construído para ser mantido — por nós ou por quem vier depois.

Aceitamos projetos de consultoria pontual, desenvolvimento end-to-end e retainer técnico para squads internas que precisam de suporte de arquitetura.
